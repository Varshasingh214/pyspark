How to setup Git:
1. Download git for windows: https://git-scm.com/download/win
2. Install git and launch git bash
3. Create a directory (mkdir <dir-name>
4. Initialize git (git init)
5. Add a file to the directory
6. Check that the file presence is captured by git or not (git status)
